About The Website

The illustrated Website was succesfully designed by Kumar Kshitij & Saksham Singh for the 5th Sems Mini Project.
This website is a showcase website, that simply features boot designs from a hypothetical brand named 'Jackem'
Jackem designs the sketch for Boots/ Sneakers/ Shoes, and sends these designs to the top brands, and these
brands then select the sketches and make an actual shoe/ sneakers/ boot out of it. This website only features a few
selections by these brands.

This website is designed under material design concept of google, all the animations are designed with javascript
and materialize transitions. All the images, extensions used in this project are open-source.

Check out all Images & Extension used in this project here!

Note: This website design is not a downloaded template, the whole code is written directly by Kumar Kshitij
& Saksham Singh.
